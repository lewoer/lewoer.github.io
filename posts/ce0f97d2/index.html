Annotations

发表于 2019-02-14 | 分类于 the complete java developer course | | 热度 ℃

Annotations: 一种元数据形式, 提供关于程序的信息(非程序自身),对代码执行没有直接影响

三种使用方式:

  • Information for the compiler: 被编译器使用检测error, 或禁止waring.
  • Compile-time and deployment-time processing: 一些软件处理注解数据以生成代码,XML文件…
  • Runtime processing: 一些注解可以在运行的时候用于检查.
